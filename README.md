# Studio-Aurora - Em Construção

<h2>Sobre</h2>
<p>Estudos de HTML e CSS.</p>
<p>Foi pedido para que a IA simulasse um pedido, criando um cliente e seus objetivos para a criação de um site, foram passados nome, seções de tela, cores e fontes. A partir disso deu-se início a criação do layout personalizado no figma, posteriormente alinhado com a IA.</p>
<p>Aplicação de design criado no figma em código. Práticas de clip-path, box-shadow, personalização, gradientes, ::before e ::after, efeitos de animação e transição.</p>
<br>
<strong>Objetivo: </strong><p>Desenvolver o básico bem-feito.</p>
<br>
<a hrref="https://brandoline.github.io/Studio-Aurora/">Clique aqui para ver o andamento do site!</a>

<h2>Comentário sobre uso de IA para estudos</h2>
<p>Nesse caso de estudo, foi utilizado IA para dois pontos essênciais: Criação de demanda e criação de objetos personalizados. Exemplo: A cor do gradiente aplicado no figma, foi passado para que a IA replicasse a composição.</p>
<p>Uma outra utilização foi através do site <a href="https://fundamentos-css.netlify.app/">Fundamentos CSS</a>, criado inteiramente por IA para consulta de estilização e uso didático, seu repositório se encontra presente no perfil para consulta.</p>
